# Data Science Projects

This repository contains a collection of analytical and machine learning projects developed to demonstrate practical problem‑solving, data interpretation, and model‑building capabilities. The work spans exploratory analysis, predictive modeling, dashboard development, and business intelligence reporting.

---

## Project Index

### Customer Segmentation Using Credit Card Data Via K-Means Clustering
**File:** `Customer_Segmetation.ipynb`  
**Summary:**  
This project focuses on customer segmentation using credit card usage data. The goal is to group credit card users based on their spending behavior, credit utilization patterns, and payment habits. By identifying customer clusters, financial institutions can design better marketing strategies, customize credit card products, and improve customer retention.

---

### Predicting Energy Needs Of Buildings
**File:** `Energy_Regression.ipynb`  
**Summary:**  
Predicting two variables Heating Load and Cooling Load of different Buildings based on their dimensions and Materials is the objective of this project. The Linear Regression model is used for prediction in this project.

---

### Market Basket Analysis of a Grocery Store
**File:** `Grocery_Dataset_Market_Basket_Analysis.ipynb`  
**Summary:**  
Executed Market Basket Analysis by applying Apriori and association rule algorithms to a grocery store's transactional datasets, extracting high-value item affinities and translating these patterns into actionable recommendations for operational and merchandising decisions.

---

### Hospital Operations & Patient Analytics Dashboard
**File:** `Healthcare_Dataset_Insights_Dashboard.pbix`  
**Summary:**  
Analyzed hospital operations, patient distribution, and resource utilization and built a Dashboard in Power BI to gain insights into hospital performance and to support decision-making.

---

### Classifying Netflix Title Types
**File:** `Netflix_Type_Classification.ipynb`  
**Summary:**  
This project deals with classifying the target column type of the dataset using Random Forest Classifier. Here, we deal with everything starting from Data cleaning to deriving and explaining meaningful insights. 

---

### Sales & Profit Analytics Dashboard
**File:** `Superstore_Dataset_Insights_Dashboard.xlsx`  
**Summary:**  
Analyzed sales performance, profitability, and customer trends and built a Dashboard in Excel to visualize key trends & insights and to support decision-making.

---

### Player Performance & Career Analytics Dashboard
**File:** `Test_Match_Dataset_Insights_Dashboard.twb`  
**Summary:**  
Built a Dashboard in Tableau to gain insights into performance trends, consistency & impact and to support decision-making amongst selectors, analysts and fans.

---
### Predicting EV Battery Blast Risks Using Sensors and Design Parameters
**File:** `XGBoost_Blast_Prediction.ipynb`  
**Summary:**  
Implemented an XGBoost model to predict blast outcomes using sensors and design parameters. The project demonstrates Machine Learning, EDA, Data Wrangling techniques.
