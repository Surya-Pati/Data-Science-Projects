# Player Performance & Career Analytics Dashboard

This project delivers an interactive Tableau dashboard to visualize match statistics, player performance, and outcome trends in test cricket. The objective is to provide sports analysts and enthusiasts with actionable insights into player contributions and career dynamics.

---

## Project Overview

Cricket generates extensive data across matches, players, and outcomes. By leveraging Tableau, this project transforms raw match data into interactive dashboards that highlight performance trends.  
The dashboard enables stakeholders to explore match outcomes, player statistics, and historical patterns in test cricket.

---

## Objectives

- Build a Tableau dashboard to visualize test match statistics  
- Analyze player performance across batting metrics  
- Provide interactive filters for dynamic exploration  
- Support sports analytics for selectors, fans & analysts

---

## Dataset

- **Source:** Test match dataset (publicly available or provided)  
- **Features:** Player statistics, match outcomes, runs, wickets, overs, and team performance metrics  
- **Target:** Sports insights through dashboard visualization

---

## Methodology

1. **Data Preparation**  
   - Cleaning and formatting raw match data  
   - Creating calculated fields for performance metrics  

2. **Dashboard Development**  
   - Interactive charts and tables  
   - Filters and parameters for dynamic exploration  
   - KPI cards for quick performance tracking  

3. **Insights**  
   - Player performance trends across matches  
   - Team outcome analysis  
   - Historical patterns in test cricket  

---

## Results

- The dashboard provides a consolidated view of match statistics and player performance  
- Interactive filters allow users to drill down into specific matches, players, or teams  
- Insights support sports analytics, fan engagement, and performance evaluation
